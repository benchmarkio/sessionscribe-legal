Datenschutzerklärung

SessionScribe · Letzte Aktualisierung: 2. April 2026

Grundprinzip: SessionScribe verarbeitet alle Audio- und Transkriptionsdaten ausschließlich auf Ihrem Gerät. Keine Audioaufnahmen, Transkripte oder personenbezogene Daten werden jemals an externe Server übermittelt.

1. Daten, die wir verarbeiten

SessionScribe nimmt Audio über das Mikrofon Ihres Geräts auf und wandelt es mithilfe von maschinellen Lernmodellen auf dem Gerät (WhisperKit und CoreML) in Text um. Die folgenden Daten werden lokal auf Ihrem Gerät erstellt und gespeichert:

  • Audioaufnahmen — werden in der App-Sandbox gespeichert, bis Sie sie löschen
  • Transkripte und Sprecherkennzeichnungen — werden in der lokalen Datenbank der App gespeichert
  • Sitzungsmetadaten (Titel, Datum, Dauer, Tags) — werden lokal gespeichert
  • App-Einstellungen (Sprache, Anzeigeeinstellungen, Abonnementstatus) — werden in UserDefaults auf Ihrem Gerät gespeichert

2. Daten, die wir nicht erheben

Wir erheben, übermitteln oder speichern keine Daten auf externen Servern. Im Einzelnen:

  • Keine Audioaufnahmen werden in die Cloud gesendet
  • Keine Transkripte verlassen Ihr Gerät
  • Keine personenbezogenen Daten werden erhoben
  • Keine Analysedaten werden an Dritte weitergegeben
  • Keine Werbe-Identifikatoren werden verwendet
  • Keine Tracking-Technologien werden eingesetzt

3. Optionale anonyme Nutzungsstatistiken

Wenn Sie in den Einstellungen „Anonyme Nutzungsstatistiken teilen" aktivieren, speichert die App aggregierte Nutzungsdaten (wie Sitzungsanzahl und insgesamt transkribierte Minuten) lokal auf Ihrem Gerät. Diese Daten werden ausschließlich zur Anzeige Ihres persönlichen Nutzungs-Dashboards verwendet. Sie werden niemals an einen Server übermittelt.

4. Mikrofon und Spracherkennung

SessionScribe benötigt Zugriff auf das Mikrofon, um Audio aufzunehmen. Die App kann außerdem die Spracherkennungsberechtigung für Apples geräteinterne Sprachverarbeitung auf unterstützten Geräten (iOS 26+) verwenden. Beide Berechtigungen werden ausschließlich für die geräteinterne Transkription genutzt. Sie können diese Berechtigungen jederzeit in den iOS-Einstellungen widerrufen.

5. Abonnements und Käufe

SessionScribe bietet optionale Abonnements (Plus und Pro) über Apples In-App-Kaufsystem an. Die gesamte Zahlungsabwicklung erfolgt durch Apple. Wir erhalten, speichern oder haben keinen Zugang zu Ihren Zahlungsinformationen. Der Abonnementstatus wird über Apples StoreKit-Framework auf Ihrem Gerät verifiziert.

6. Face ID / Biometrische Authentifizierung

Wenn Sie die optionale App-Sperre aktivieren, verwendet SessionScribe Apples LocalAuthentication-Framework zur Identitätsprüfung. Biometrische Daten (Face ID oder Touch ID) werden ausschließlich von iOS verarbeitet und sind für die App zu keinem Zeitpunkt zugänglich.

7. Datenspeicherung und Löschung

Alle Daten werden ausschließlich auf Ihrem Gerät gespeichert. Sie können einzelne Aufnahmen und Transkripte jederzeit innerhalb der App löschen. Durch die Deinstallation der App werden alle zugehörigen Daten endgültig entfernt.

8. Datenschutz für Kinder

SessionScribe richtet sich nicht an Kinder unter 13 Jahren. Wir erheben wissentlich keine Daten von Kindern.

9. Drittanbieterdienste

SessionScribe integriert keine Analyse-, Werbe- oder Tracking-SDKs von Drittanbietern. Die einzige Drittanbieterkomponente ist die WhisperKit-Bibliothek für maschinelles Lernen, die vollständig auf Ihrem Gerät ausgeführt wird.

10. Änderungen dieser Datenschutzerklärung

Wir können diese Datenschutzerklärung von Zeit zu Zeit aktualisieren. Das Datum „Letzte Aktualisierung" oben gibt die jeweils neueste Fassung an. Die fortgesetzte Nutzung der App nach Änderungen gilt als Zustimmung zur aktualisierten Datenschutzerklärung.
